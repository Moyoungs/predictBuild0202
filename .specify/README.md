## 🚀 로컬에서 이어서 개발하는 방법

### 프로토타입 바로 돌려보기

```bash
cd prototype
pip install openpyxl pyyaml xlrd
python reporter.py  # 샘플 실행
```

**주의:** `prototype/*.py` 의 파일 경로는 샌드박스 절대 경로로 하드코딩되어 있습니다.
로컬에서는 파일 경로를 본인 환경에 맞게 수정하세요.

### SpecKit으로 정식 프로젝트 시작

```bash
# 1. 새 프로젝트 스캐폴딩
uv init quantity-aggregator
cd quantity-aggregator

# 2. SpecKit 문서 배치
mkdir -p .specify
cp /path/to/speckit/*.md .specify/

# 3. 사전 배치
mkdir -p data
cp /path/to/data/terminology.yaml data/

# 4. tasks.md의 Phase 0부터 체크리스트 따라가기
```
